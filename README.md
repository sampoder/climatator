**Planned features:**

* Rumbling chair

* Heating (moves closer to person)
  * Consumer heater, on pulley system
  * Rising temp, more heat waves
* Water sprayer

  * Rising sea levels, unpredicatable rain
* Big screen
  * Old TV
* Wind
  * Fan on a smart plug
  * Stronger hurricanes
* Surround sound
  * Multiple speakers
* Dangling food
  * Pulley system
